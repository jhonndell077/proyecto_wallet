RESUMEN

¿Qué es Income Wallet?

Un sistema integral de control financiero diseñado para negocios modernos.

🎯 Propósito

Income Wallet es una aplicación web para gestionar ingresos, gastos y cuentas por cobrar con análisis avanzado y visualización de datos en tiempo real.

👥 Público Objetivo

Pequeñas y medianas empresas, restaurantes, tiendas de retail, profesionales independientes y cualquier negocio que necesite control financiero.

⚡ Ventajas Principales

Interfaz moderna, persistencia local, sin dependencia de nube, análisis de márgenes, exportación de datos y migración fácil desde otros sistemas.

MÓDULOS

Funcionalidades del Sistema

Cada componente de Income Wallet explicado en detalle.

1. Dashboard - Resumen

El centro de control de Income Wallet donde se presenta una visión general del estado financiero actual del negocio.

Funcionalidades principales:

  • 📊 4 KPIs principales: Ingresos cobrados, Gastos totales, Cuentas por cobrar, Balance neto
  • 📈 Gráfico de flujo de caja: Visualización de los últimos 7 días con ingresos y gastos
  • ⚡ Atajos rápidos: Botones para acciones comunes (nuevo ingreso, gasto, ver cobros)
  • 💡 Insights automáticos: Análisis inteligente del estado financiero actual
  • 📱 Widget lateral: Balance actual y cobros pendientes siempre visibles
📊 Gráfico de Flujo de Caja (7 días)

2. Registro de Movimientos

Formulario inteligente para capturar nuevos ingresos y gastos con toda la información necesaria.

Campos del formulario:

  • 💰 Tipo: Ingreso o Gasto (selector)
  • 📅 Fecha: Selector de fecha (default: hoy)
  • Categoría: Texto libre para clasificación personalizada
  • 💳 Método de pago: Efectivo, Transferencia, Tarjeta, Crédito
  • 💵 Monto: Numérico con 2 decimales
  • 👤 Cliente: Nombre opcional (para seguimiento)
  • 📞 Teléfono: Contacto opcional del cliente
  • Descripción: Detalles adicionales del movimiento
  • ✅ Cobrado/Pagado: Checkbox para estado inmediato

3. Cuentas por Cobrar

Gestión especializada para ventas a crédito con seguimiento de vencimientos y alertas visuales.

Funcionalidades:

  • 📋 Lista completa: Todas las cuentas pendientes con detalles
  • 👤 Información del cliente: Nombre y teléfono para contacto
  • 📅 Fecha de vencimiento: Claramente visible en cada cuenta
  • 🔔 Alertas visuales: Cuentas próximas a vencer destacadas
  • ✅ Marcar como cobrado: Acción rápida para cerrar cuentas
  • 📊 Total pendiente: Suma de todas las cuentas por cobrar

4. Historial de Movimientos

Registro completo de todas las transacciones con capacidades de búsqueda, filtrado y edición.

Funcionalidades:

  • Búsqueda: Por texto, categoría o cliente
  • 🎯 Filtrado: Por tipo (ingresos/gastos)
  • 📊 Vista detallada: Información completa de cada movimiento
  • Edición: Modificar registros existentes
  • 🗑 Eliminación: Remover movimientos erróneos
  • 📅 Ordenamiento: Por fecha (más reciente primero)

5. Análisis de Márgenes

Herramienta avanzada para analizar el rendimiento del negocio por períodos de tiempo.

Funcionalidades:

  • 📊 Selección de período: Año y mes específicos
  • 📈 Mejor mes: Identificación automática del período más productivo
  • 📉 Peor mes: Detección de períodos críticos
  • 💰 Cálculo de márgenes: Rentabilidad por período
  • 📋 Reporte detallado: Tabla con todos los movimientos del período
  • 🎯 Insights: Conclusiones automáticas para toma de decisiones
📊 Análisis de Tendencias y Márgenes

6. Configuración

Panel de administración para personalizar el sistema y gestionar datos.

Funcionalidades:

  • Nombre del negocio: Personalización de la marca
  • 💾 Exportar JSON: Backup completo de datos
  • 📥 Importar JSON: Restauración de backup
  • 🔄 Migración desde Wally: Importación desde sistemas anteriores
  • 🧪 Datos de prueba: Generar datos demo para testing
  • 🗑 Reset de datos: Limpiar todos los datos del sistema

GUA DE USO

Cómo Usar Income Wallet

Instrucciones paso a paso para aprovechar al máximo el sistema.

1

Instalación Inicial

Abre el archivo index.html en tu navegador moderno. No requiere instalación ni servidor. PulsaFlow funciona completamente en el cliente.

2

Configuración del Negocio

Ve a la sección Configuración y establece el nombre de tu negocio. Personaliza la etiqueta según tus preferencias.

3

Registrar Primer Movimiento

Ve a "Nuevo movimiento", selecciona el tipo (ingreso/gasto), completa los campos y guarda. El movimiento aparecerá inmediatamente en el Dashboard.

4

Revisar el Dashboard

Observa cómo los KPIs se actualizan automáticamente. El gráfico de flujo de caja mostrará la tendencia de tus movimientos.

5

Gestionar Cuentas por Cobrar

Para ventas a crédito, desmarca "cobrado al momento" y establece una fecha de cobro. La cuenta aparecerá en la sección "Por cobrar".

6

Analizar Resultados

Usa el módulo de "Análisis de márgenes" para identificar tus mejores meses y tomar decisiones estratégicas basadas en datos.

7

Backup Regular

En Configuración, usa "Exportar JSON" regularmente para mantener copias de seguridad de tus datos importantes.
